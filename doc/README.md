# VisionAutoTest Docs

`doc/` 目录是 VisionAutoTest 的项目文档中心，主要承载以下几类内容：

- 产品定位与阶段目标
- 前后端架构设计
- API 与数据库契约
- 实现规范与重构治理文档
- 联调与验收资料

## 入口建议

- 如果你想快速了解项目全貌：看 [文档索引](./文档索引.md)
- 如果你是 PM 或项目负责人：先看 [PRD](./企业级跨端视觉自动化测试平台PRD.md) 和 [MVP 定义](./MVP定义.md)
- 如果你是研发：从 [文档索引](./文档索引.md) 进入对应架构、API 和数据库文档
- 如果你是联调或测试：优先看 [联调文档](./联调文档/00-前端联调总览.md)

## 当前整理原则

当前 `doc/` 目录只保留长期有效、可持续维护的项目文档。

以下内容原则上不再长期保留在这里：

- 一次性迭代需求单
- 临时技术统筹单
- 仅服务单次协作过程的说明文档
- 与 README 或环境样例重复、且已过时的说明文档

如后续新增文档，建议优先放入以下类别之一：

- 产品文档
- 架构文档
- API 契约文档
- 数据库文档
- 实现规范与治理文档
- 联调文档
