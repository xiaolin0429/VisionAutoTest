# Frontend Docs

本目录存放 VisionAutoTest 前端相关的长期文档，面向前端研发、Tech Lead、联调和页面级交互治理场景。

## 适合什么时候读

- 你要理解前端整体架构、路由和状态管理方式
- 你要新增或修改页面、组件、前端 API 模块、类型映射
- 你要核对前端页面与后端契约是否一致
- 你要做前端联调、页面 review 或交互治理

## 推荐阅读顺序

1. [前端技术架构设计文档](./前端技术架构设计文档.md)

## 文档分工

- `前端技术架构设计文档.md`
  说明前端技术选型、核心模块、状态流转、工作空间隔离和关键交互链路。

## 当前维护原则

本目录只保留长期有效的前端架构与规范文档。

如果后续出现阶段性的页面改版方案、一次性联调说明或短期实现草案，原则上不直接长期沉淀在这里，而应按内容性质选择：

- 进入 `doc/联调文档/`
- 沉淀为长期规范后再进入 `doc/frontend/`
- 落到 `.claude/skills/visionautotest-frontend-standard-flow/SKILL.md`
