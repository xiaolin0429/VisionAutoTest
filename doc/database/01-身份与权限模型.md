# 身份与权限模型

## 1. 表清单

| 表名 | 说明 |
|---|---|
| `iam_users` | 用户主表 |
| `iam_roles` | 角色表 |
| `iam_permissions` | 权限点表 |
| `iam_role_permissions` | 角色权限关联表 |
| `iam_user_roles` | 用户与系统角色关联表 |
| `iam_sessions` | 登录会话表 |
| `iam_refresh_tokens` | 刷新令牌表 |

## 2. 表设计

### 2.1 iam_users

用途：存储平台用户主体。

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| `id` | BIGINT | PK | 主键 |
| `username` | VARCHAR(64) | NOT NULL, UK | 登录名，全局唯一 |
| `display_name` | VARCHAR(128) | NOT NULL | 展示名 |
| `email` | VARCHAR(128) | NULL | 邮箱 |
| `mobile` | VARCHAR(32) | NULL | 手机号 |
| `password_hash` | VARCHAR(255) | NOT NULL | 密码哈希 |
| `status` | VARCHAR(32) | NOT NULL | `active/inactive/locked` |
| `last_login_at` | TIMESTAMP | NULL | 最近登录时间 |
| `created_at` | TIMESTAMP | NOT NULL | 创建时间 |
| `created_by` | BIGINT | NULL | 创建人 |
| `updated_at` | TIMESTAMP | NOT NULL | 更新时间 |
| `updated_by` | BIGINT | NULL | 更新人 |
| `version_no` | INTEGER | NOT NULL DEFAULT 1 | 乐观锁 |
| `is_deleted` | BOOLEAN | NOT NULL DEFAULT FALSE | 逻辑删除 |

索引建议：

- `uk_iam_users_username(username, is_deleted)`
- `idx_iam_users_status(status)`

### 2.2 iam_roles

用途：定义系统级角色。

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| `id` | BIGINT | PK | 主键 |
| `role_code` | VARCHAR(64) | NOT NULL, UK | 角色编码 |
| `role_name` | VARCHAR(128) | NOT NULL | 角色名称 |
| `scope_type` | VARCHAR(32) | NOT NULL | `system/workspace` |
| `description` | VARCHAR(512) | NULL | 描述 |
| `status` | VARCHAR(32) | NOT NULL | 状态 |
| `created_at` | TIMESTAMP | NOT NULL | 创建时间 |
| `created_by` | BIGINT | NULL | 创建人 |
| `updated_at` | TIMESTAMP | NOT NULL | 更新时间 |
| `updated_by` | BIGINT | NULL | 更新人 |
| `version_no` | INTEGER | NOT NULL DEFAULT 1 | 乐观锁 |
| `is_deleted` | BOOLEAN | NOT NULL DEFAULT FALSE | 逻辑删除 |

### 2.3 iam_permissions

用途：权限点字典表。

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| `id` | BIGINT | PK | 主键 |
| `permission_code` | VARCHAR(128) | NOT NULL, UK | 权限编码 |
| `permission_name` | VARCHAR(128) | NOT NULL | 权限名称 |
| `module_name` | VARCHAR(64) | NOT NULL | 模块名 |
| `description` | VARCHAR(512) | NULL | 描述 |

### 2.4 iam_role_permissions

用途：角色与权限点的多对多关系表。

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| `id` | BIGINT | PK | 主键 |
| `role_id` | BIGINT | NOT NULL, FK | 角色 ID |
| `permission_id` | BIGINT | NOT NULL, FK | 权限 ID |
| `created_at` | TIMESTAMP | NOT NULL | 创建时间 |

约束与索引：

- 唯一约束：`uk_iam_role_permissions(role_id, permission_id)`
- 索引：`idx_iam_role_permissions_permission_id(permission_id)`

### 2.5 iam_user_roles

用途：用户与系统角色的多对多关系表。

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| `id` | BIGINT | PK | 主键 |
| `user_id` | BIGINT | NOT NULL, FK | 用户 ID |
| `role_id` | BIGINT | NOT NULL, FK | 角色 ID |
| `created_at` | TIMESTAMP | NOT NULL | 创建时间 |
| `created_by` | BIGINT | NULL | 创建人 |

约束与索引：

- 唯一约束：`uk_iam_user_roles(user_id, role_id)`
- 索引：`idx_iam_user_roles_role_id(role_id)`

### 2.6 iam_sessions

用途：登录会话与访问令牌主体的持久化锚点，用于支持 `sessions/current` 查询与主动吊销。

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| `id` | BIGINT | PK | 主键 |
| `user_id` | BIGINT | NOT NULL, FK | 用户 ID |
| `session_code` | VARCHAR(64) | NOT NULL, UK | 会话编码 |
| `access_token_jti` | VARCHAR(64) | NOT NULL, UK | 当前访问令牌标识 |
| `client_type` | VARCHAR(32) | NOT NULL | `web/api/ci` |
| `client_ip` | VARCHAR(64) | NULL | 客户端 IP |
| `user_agent` | VARCHAR(512) | NULL | 客户端 UA |
| `status` | VARCHAR(32) | NOT NULL | `active/revoked/expired` |
| `last_seen_at` | TIMESTAMP | NULL | 最近活跃时间 |
| `expires_at` | TIMESTAMP | NOT NULL | 会话过期时间 |
| `revoked_at` | TIMESTAMP | NULL | 吊销时间 |
| `created_at` | TIMESTAMP | NOT NULL | 创建时间 |
| `updated_at` | TIMESTAMP | NOT NULL | 更新时间 |

约束与索引：

- 唯一约束：`uk_iam_sessions_session_code(session_code)`
- 唯一约束：`uk_iam_sessions_access_token_jti(access_token_jti)`
- 索引：`idx_iam_sessions_user_status(user_id, status)`

### 2.7 iam_refresh_tokens

用途：刷新令牌轮换与吊销控制表。

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| `id` | BIGINT | PK | 主键 |
| `session_id` | BIGINT | NOT NULL, FK | 会话 ID |
| `token_hash` | VARCHAR(255) | NOT NULL, UK | 刷新令牌哈希 |
| `status` | VARCHAR(32) | NOT NULL | `active/used/revoked/expired` |
| `expires_at` | TIMESTAMP | NOT NULL | 过期时间 |
| `consumed_at` | TIMESTAMP | NULL | 被续期使用时间 |
| `revoked_at` | TIMESTAMP | NULL | 吊销时间 |
| `created_at` | TIMESTAMP | NOT NULL | 创建时间 |

约束与索引：

- 唯一约束：`uk_iam_refresh_tokens_token_hash(token_hash)`
- 索引：`idx_iam_refresh_tokens_session_status(session_id, status)`

## 3. 设计说明

- 用户与工作空间关系不在本模块定义，统一放到工作空间模型中。
- 角色不直接内嵌权限 JSON，使用关系表保持可治理性。
- 用户与系统角色关系使用 `iam_user_roles` 维护，避免在 `iam_users` 中塞入多值字段。
- 当前会话、令牌续期与吊销能力分别由 `iam_sessions`、`iam_refresh_tokens` 承载。
- 密码只存哈希，不存任何可逆明文。
