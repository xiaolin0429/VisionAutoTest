# 模板与基准资产模型

## 1. 表清单

| 表名 | 说明 |
|---|---|
| `asset_templates` | 模板主表 |
| `asset_template_mask_regions` | 模板忽略区域表 |
| `asset_baseline_revisions` | 模板基准版本表 |

## 2. 表设计

### 2.1 asset_templates

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| `id` | BIGINT | PK | 主键 |
| `workspace_id` | BIGINT | NOT NULL, FK | 工作空间 ID |
| `template_code` | VARCHAR(64) | NOT NULL | 模板编码 |
| `template_name` | VARCHAR(128) | NOT NULL | 模板名称 |
| `template_type` | VARCHAR(32) | NOT NULL | `page/component/text_region` |
| `match_strategy` | VARCHAR(32) | NOT NULL | 当前版本仅支持 `template/ocr` |
| `threshold_value` | DECIMAL(5,4) | NOT NULL | 匹配阈值 |
| `status` | VARCHAR(32) | NOT NULL | `draft/published/archived` |
| `current_baseline_revision_id` | BIGINT | NULL | 当前基准版本 ID |
| `created_at` | TIMESTAMP | NOT NULL | 创建时间 |
| `created_by` | BIGINT | NULL | 创建人 |
| `updated_at` | TIMESTAMP | NOT NULL | 更新时间 |
| `updated_by` | BIGINT | NULL | 更新人 |
| `version_no` | INTEGER | NOT NULL DEFAULT 1 | 乐观锁 |
| `is_deleted` | BOOLEAN | NOT NULL DEFAULT FALSE | 逻辑删除 |

约束与索引：

- 唯一约束：`uk_asset_templates(workspace_id, template_code, is_deleted)`
- 索引：`idx_asset_templates_workspace_type(workspace_id, template_type)`

### 2.2 asset_template_mask_regions

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| `id` | BIGINT | PK | 主键 |
| `template_id` | BIGINT | NOT NULL, FK | 模板 ID |
| `region_name` | VARCHAR(128) | NOT NULL | 区域名称 |
| `x_ratio` | DECIMAL(8,6) | NOT NULL | 横向起点比例 |
| `y_ratio` | DECIMAL(8,6) | NOT NULL | 纵向起点比例 |
| `width_ratio` | DECIMAL(8,6) | NOT NULL | 宽比例 |
| `height_ratio` | DECIMAL(8,6) | NOT NULL | 高比例 |
| `sort_order` | INTEGER | NOT NULL DEFAULT 1 | 排序 |
| `created_at` | TIMESTAMP | NOT NULL | 创建时间 |
| `updated_at` | TIMESTAMP | NOT NULL | 更新时间 |

约束建议：

- `x_ratio`、`y_ratio` 必须在 `[0, 1)` 区间内
- `width_ratio`、`height_ratio` 必须在 `(0, 1]` 区间内
- 必须满足 `x_ratio + width_ratio <= 1` 且 `y_ratio + height_ratio <= 1`

### 2.3 asset_baseline_revisions

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| `id` | BIGINT | PK | 主键 |
| `template_id` | BIGINT | NOT NULL, FK | 模板 ID |
| `revision_no` | INTEGER | NOT NULL | 版本号 |
| `media_object_id` | BIGINT | NOT NULL, FK | 基准图媒体对象 ID |
| `source_type` | VARCHAR(32) | NOT NULL | `manual/uploaded/adopted_from_failure` |
| `is_current` | BOOLEAN | NOT NULL DEFAULT FALSE | 是否当前版本 |
| `remark` | VARCHAR(512) | NULL | 备注 |
| `created_at` | TIMESTAMP | NOT NULL | 创建时间 |
| `created_by` | BIGINT | NULL | 创建人 |

约束与索引：

- 唯一约束：`uk_asset_baseline_revisions(template_id, revision_no)`
- 需要使用“仅对 `is_current = true` 生效”的部分唯一索引或等价实现，保证同一模板最多只有一个当前版本

## 3. 设计说明

- 媒体对象元数据与生命周期已拆分到 [08-媒体对象与引用模型](./08-媒体对象与引用模型.md)。
- 忽略区域不存像素绝对值，统一存相对比例，保障跨分辨率稳定。
- 基准版本采用独立表，满足历史追溯和一键回滚。
- 当前版本仅 `published` 模板允许进入执行链路；`draft` 与 `archived` 不能作为可执行模板。
- `orb`、`mixed` 为后续阶段能力，不纳入当前版本可执行策略集合。
