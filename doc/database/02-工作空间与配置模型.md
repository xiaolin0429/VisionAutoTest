# 工作空间与配置模型

## 1. 表清单

| 表名 | 说明 |
|---|---|
| `core_workspaces` | 工作空间主表 |
| `core_workspace_members` | 工作空间成员表 |
| `cfg_environment_profiles` | 环境档案表 |
| `cfg_environment_variables` | 环境变量表 |
| `cfg_device_profiles` | 设备预设表 |

## 2. 表设计

### 2.1 core_workspaces

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| `id` | BIGINT | PK | 主键 |
| `workspace_code` | VARCHAR(64) | NOT NULL, UK | 工作空间编码 |
| `workspace_name` | VARCHAR(128) | NOT NULL | 工作空间名称 |
| `description` | VARCHAR(512) | NULL | 描述 |
| `status` | VARCHAR(32) | NOT NULL | `active/inactive` |
| `owner_user_id` | BIGINT | NOT NULL, FK | 拥有者 |
| `created_at` | TIMESTAMP | NOT NULL | 创建时间 |
| `created_by` | BIGINT | NULL | 创建人 |
| `updated_at` | TIMESTAMP | NOT NULL | 更新时间 |
| `updated_by` | BIGINT | NULL | 更新人 |
| `version_no` | INTEGER | NOT NULL DEFAULT 1 | 乐观锁 |
| `is_deleted` | BOOLEAN | NOT NULL DEFAULT FALSE | 逻辑删除 |

### 2.2 core_workspace_members

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| `id` | BIGINT | PK | 主键 |
| `workspace_id` | BIGINT | NOT NULL, FK | 工作空间 ID |
| `user_id` | BIGINT | NOT NULL, FK | 用户 ID |
| `workspace_role` | VARCHAR(64) | NOT NULL | `workspace_admin/workspace_member` |
| `status` | VARCHAR(32) | NOT NULL | 成员状态 |
| `joined_at` | TIMESTAMP | NOT NULL | 加入时间 |
| `created_at` | TIMESTAMP | NOT NULL | 创建时间 |

约束与索引：

- 唯一约束：`uk_core_workspace_members(workspace_id, user_id)`
- 索引：`idx_core_workspace_members_user_id(user_id)`

### 2.3 cfg_environment_profiles

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| `id` | BIGINT | PK | 主键 |
| `workspace_id` | BIGINT | NOT NULL, FK | 工作空间 ID |
| `profile_name` | VARCHAR(128) | NOT NULL | 环境名称 |
| `base_url` | VARCHAR(255) | NOT NULL | 基础地址 |
| `description` | VARCHAR(512) | NULL | 描述 |
| `status` | VARCHAR(32) | NOT NULL | 状态 |
| `created_at` | TIMESTAMP | NOT NULL | 创建时间 |
| `created_by` | BIGINT | NULL | 创建人 |
| `updated_at` | TIMESTAMP | NOT NULL | 更新时间 |
| `updated_by` | BIGINT | NULL | 更新人 |
| `version_no` | INTEGER | NOT NULL DEFAULT 1 | 乐观锁 |
| `is_deleted` | BOOLEAN | NOT NULL DEFAULT FALSE | 逻辑删除 |

约束与索引：

- 唯一约束：`uk_cfg_environment_profiles(workspace_id, profile_name, is_deleted)`

### 2.4 cfg_environment_variables

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| `id` | BIGINT | PK | 主键 |
| `environment_profile_id` | BIGINT | NOT NULL, FK | 环境档案 ID |
| `var_key` | VARCHAR(128) | NOT NULL | 变量键 |
| `var_value_ciphertext` | TEXT | NOT NULL | 加密后的变量值 |
| `is_secret` | BOOLEAN | NOT NULL DEFAULT FALSE | 是否敏感 |
| `description` | VARCHAR(512) | NULL | 描述 |
| `created_at` | TIMESTAMP | NOT NULL | 创建时间 |
| `updated_at` | TIMESTAMP | NOT NULL | 更新时间 |

约束与索引：

- 唯一约束：`uk_cfg_environment_variables(environment_profile_id, var_key)`

### 2.5 cfg_device_profiles

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| `id` | BIGINT | PK | 主键 |
| `workspace_id` | BIGINT | NOT NULL, FK | 工作空间 ID |
| `profile_name` | VARCHAR(128) | NOT NULL | 设备名称 |
| `device_type` | VARCHAR(32) | NOT NULL | `web/mobile/tablet` |
| `viewport_width` | INTEGER | NOT NULL | 视口宽 |
| `viewport_height` | INTEGER | NOT NULL | 视口高 |
| `device_scale_factor` | DECIMAL(6,2) | NOT NULL | 缩放因子 |
| `user_agent` | VARCHAR(512) | NULL | UA |
| `is_default` | BOOLEAN | NOT NULL DEFAULT FALSE | 是否默认 |
| `created_at` | TIMESTAMP | NOT NULL | 创建时间 |
| `created_by` | BIGINT | NULL | 创建人 |
| `updated_at` | TIMESTAMP | NOT NULL | 更新时间 |
| `updated_by` | BIGINT | NULL | 更新人 |
| `version_no` | INTEGER | NOT NULL DEFAULT 1 | 乐观锁 |
| `is_deleted` | BOOLEAN | NOT NULL DEFAULT FALSE | 逻辑删除 |

## 3. 设计说明

- 工作空间成员关系单独建表，不把成员列表存为 JSON。
- 环境变量采用明细表，避免环境档案表字段膨胀。
- 敏感变量默认加密，接口层按权限脱敏。
