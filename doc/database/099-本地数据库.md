# 本地数据库

## 1. PostgreSQL 本地连接信息

- 数据库类型：`PostgreSQL`
- 账号：`postgres`
- 密码：`postgres`
- 端口：`5432`
- 管理库：`postgres`
- 开发库：`AutoTestDev`

## 2. 本地开发使用约定

- `AutoTestDev` 仅用于本地开发调试与自测
- 应用默认业务库连接为：

```text
postgresql+psycopg://postgres:postgres@127.0.0.1:5432/AutoTestDev
```

- 若 `AutoTestDev` 不存在，可由初始化命令或开发环境启动流程自动创建

## 3. 测试数据库说明

- 测试数据库不使用 `AutoTestDev`
- 测试阶段需单独准备独立测试库
- 应用和测试代码只负责连接并迁移测试库，不负责自动创建测试数据库
